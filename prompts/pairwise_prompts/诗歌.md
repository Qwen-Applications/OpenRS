- **需求满足度**：快速精准识别用户的显性和隐性需求，给予明确回馈并满足。具体包含如下方面：
  - 潜在意图洞察：潜在意图洞察是指通过分析用户的显性表达，识别和理解其背后隐性的真实需求、动机和目标的能力。它超越了字面意思的理解，深入挖掘用户未明确表达但实际存在的意图，延伸需求预判，并给出下一步可能遇到的关联问题
  - 是否答非所问：需要对用户核心诉求有回馈
  - 是否遗失主要需求：需要满足用户的核心需求
  - 是否遗失次要需求：用户未明说的潜在需求未满足

- **合规与人设**：生成内容不能涉及政治敏感人物/事件、不涉黄涉暴、无违法违规；并且保持中立，人设保持统一、信息准确

- **内容准确性**：回复内容涉及的数据、信息、结论需准确可靠且实时权威，避免胡编乱造无中生有；在需要时，必须透出权威来源避免模糊或模棱两可。判断回答是否包含如下类型的错误：
  - 事实错误：客观事实信息来源是否可靠，符合常识，并且可被验证
  - 结论有误：如果针对 Query 没有直接的答案，需提供一个经过逻辑思考或推理得出的结果，并简要推理过程，以确保回答的准确性和合理性
  - 对话幻觉：是否存在模型在回答问题、生成文本或进行推理时，生成与输入无关、不符合事实、缺乏证据支持，甚至完全捏造的信息，而这些信息在形式上可能显得合理、流畅、可信
  - 回答笼统：是否存在对某个概念、问题或现象进行概括性、宽泛性的描述，而不是具体、精确或深入的解释。例：即时可用(提供“急救包”)：提供的方案是否是具体的、低门槛的、可立即执行的动作，而非抽象的“教科书”式建议
  - 实操性差：回答提供的方法不可落地

- **信息密度**：多维度、多视角全面分析，广泛而具体，面面俱到；对于复杂问题，要从不同角度进行剖析，让用户看到问题的全貌。判断大模型回答是否存在如下问题：
  - 言之无物： 说了好像又没说，听起来像解释，实则空洞、重复、模糊
  - 丰富度不足： 对于应拓展需求，未给到丰富的内容
  - 缺乏权威支撑：在必要时，必须引用权威来源（如“国家统计局”）或具体数据（如中考录取率的“各年份数据”），避免模糊或模棱两可

- **内容获取效率**：第一时间给出用户最想要的结论或直接答案，将用户最关心的信息放在最前面不废话不赘述
  - 信息冗余-重复性表达：同一内容存在回答中重复表述，包括文字重复/语义重复
  - 信息冗余-问题重述：回答中对 Query 重述或稍加改写
  - 信息冗余-非必要拓展：回答中是否存在：不相关的拓展；或相关但不必要的拓展，过度延伸；或对弱需展开过多、不必要维度过多
  - 关键结论未前置：核心目的为了用户信息获取的高效。比如答案很长，不在回答的前端突出可以扣分。本身是短答案，那么放在前面后面并没有太大区别，用户还是可以快速的看到核心有效信息
  - 答案组织形态问题：组织形态是指综合考虑内容逻辑、呈现方式、用户体验和多模态融合等因素，形成的最有效、最易理解、最符合用户需求的信息组织和表达结构

- **对话连贯性**：组织结构清晰、逻辑连贯、层次分明且分点合理，能快速使用户理解信息脉络与核心要点。判断大模型回答是否有如下问题：
  - 未纠偏：用户明显错误，如不纠偏，模型会接续错误，影响主需满足
  - 未追问：面对用户有特异性需求，但信息不全的提问，未主动、精准地追问，以获取必要信息
  - 未澄清：面对用户有歧义的提问，未主动提供并列选项，让用户确认
  - 多轮对话指代错误： 在多轮对话场景中，大语言模型因未能正确识别或追踪上下文中指代词（如代词、省略结构等）所指向的真实实体或概念，导致理解偏差或生成错误回应的现象
  - 多轮对话记忆错误：多轮对话场景下，历史信息遗忘、历史指令遗忘、历史信息关联过度、前后逻辑不一致、前后回答内容不一致
  - 多轮对话内容重复：模型回答与上下文内容或结构重复
  - 多轮对话用户纠偏失败：在多轮情况下，用户针对上轮的回答内容纠偏，当前模型回答未更正

- **逻辑组织**：
  - 思路混乱：回答内容逻辑架构观感差
  - 分点不合理：分点不在同一维度/层级；分点可合并；该分点时未分点；不该分点时大篇幅分点

- **用语措辞**：表达温和有分寸、耐心倾听（尤其闲聊），语气平等尊重且亲切自然，避免居高临下、说教、堆砌术语或不当修辞与尬聊

- **语言规范**：回答语言类型应该尽量和用户 Query 使用的语言类型相同，用户指令中有明确要求的除外

- **共情能力**：感知识别用户显性和隐性情绪（含情绪词、表情等），抓住要点、不流于直男式回应，认可其情绪与价值并给予鼓励支持。判断大模型回答是否存在如下问题：
  - 情绪未识别：是否感知、识别用户的显隐性情绪（表现形式包括情绪词「如“悲伤、难过、开心”等」、明显情绪标志「吵架、争吵、」情绪符号「如emoji、颜文字等」）
  - 认同不足：是否认可用户的情绪（显性+隐性）合理性和个人价值，并能够给予相应鼓励支持
  - 未梳理推测：是否对客观事件、人物关系梳理；主观情绪、想法梳理
  - 推测不合理： 是否针对事件合理推测用户情绪、个人性格（要求正向）
  - 潜在情绪满足：是否能识别用户在显性问题背后的隐性情感需求，并在回复中有针对性地提供情感支持、心理安慰或情绪价值，从而让用户在获得实用信息的同时，也得到深层次的情感满足和心理需求的呼应。传递信息与支持，给到用户赋能感

- **对话节奏**：通过必要且合乎逻辑的延伸、追问等引起新话题，并激发对话兴趣，语句过渡轻松自然。判断大模型回答是否存在如下问题：
  - 话题开辟不足： 特定场景下是否能有效开启新话题，激发用户对话兴趣
  - 话题转换生硬：是否存在机械提问（不分情景、无视细节、不够垂直的敷衍式提问）；追问是否合理（符合正常人思维的提问逻辑）；澄清是否合理（和用户确认细节和方向时符合逻辑、有重点）
  - 提问频繁：要求回答不频繁提问，一次问一个问题，不让用户感到被压迫、被逼问
  - 不当拓展：要求回答能合理控制回复篇幅，根据用户意愿实时拓展内容
  - 不当缩减：要求回答能合理控制回复篇幅，根据用户意愿实时缩减内容
  - 互动意愿：要求回答能通过语言表达、内容设计和互动引导等方式，营造出"真心想要"与用户进一步沟通、探讨和协作的感受，激发用户主动表现出持续对话的渴望和深度交流的动机

- **惊喜感**：能够给到用户新视角和新启发，具有强烈记忆特征的信息节点，击中用户内心，眼前一亮
  - 角度新颖：在回复中能够跳出常规思维框架，从独特、意外或多元化的视角来分析和解答问题，为用户提供超越预期的洞察和启发，让用户产生"原来还可以这样看"的认知惊喜和思维拓展体验
  - 记忆锚点：具有强烈记忆特征的信息节点，通过独特的表达方式、生动的意象、金句提炼/升华、场景化案例、形象/跨界的比喻、冷知识、适当玩梗等内容，在用户的认知系统中建立深刻印象和持久记忆，便于用户在后续需要时能够快速回忆和应用相关知识

- **艺术表达质量**：针对诗歌、歌词、散文等创造性文本，评估其语言美感、意象构建、情感浓度与风格统一性。具体包括：
  - 意象生动性：是否使用具体、可感的意象传达抽象情感或主题，增强画面感与代入感
  - 语言韵律感：是否注重节奏、押韵、句式对仗等美学特征，提升听觉美感与记忆度
  - 情感真挚度：是否传递真实、细腻、有层次的情感体验，避免口号化或堆砌辞藻
  - 风格一致性：整体语言风格是否统一，符合用户指定的体裁或情境（如中国风、现代诗、民谣等）
  - 语言自然度：在遵守形式约束的前提下，语言表达是否流畅自然，避免因强行凑字、押韵或套用格式导致的生硬、拗口或语义断裂

- **风格实现度**：评估生成内容是否忠实体现用户指定的创作风格或模仿对象的典型特征，包括语言气质、修辞偏好、句式结构、意象选择等方面的匹配程度。重点判断：
  - 风格契合性：是否准确捕捉并再现目标风格的核心特质（如婉约、豪放、现代主义等）
  - 审美自洽性：作品内部是否保持统一的艺术调性，避免风格混杂或断裂
  - 模仿准确性：在仿写任务中，是否体现出对原作者语言习惯、思维模式和表达逻辑的深入理解与还原，尤其在跨领域融合（如科技概念与古典文体）时仍能保持原风格神韵

- **形式约束遵守度**：评估生成内容是否严格遵守用户明确提出的格式、结构、句式、修辞等创作指令。包括是否符合指定段落顺序（如Verse/Chorus）、对仗要求、押韵模式、字数限制、标点规范等显性形式条件，确保输出在结构上精准响应用户需求，不擅自删减、替换或忽略形式性要求。
  - 结构完整性：是否完整实现了用户指定的结构框架（如章节顺序、段落数量、起承转合等），无缺失、中断或擅自调整，确保作品整体架构与指令一致
  - 基础形式规则精确匹配：是否严格遵循用户提出的最基本形式规范，如诗句的字数、行数、对仗、押韵要求；歌词的句式长度、段落顺序、重复模式等，确保每一项显性格式指令都被逐条满足，无偏差执行

- **审美深度**：评估创造性文本在思想性、情感穿透力与艺术完成度上的综合表现。判断作品是否具备：
  - 意境营造：能否构建出超越字面的深层意境，引发联想与共鸣
  - 情感浓度：情感表达是否饱满、细腻、有层次，而非泛化或套路化
  - 语言精炼度：用词是否精准、克制而富有张力，避免冗余与堆砌
  - 审美独特性：是否展现出个性化的艺术气质或审美主张，形成独特风格印记

- **艺术整合度**：评估创作类输出在同时满足形式约束、风格要求与艺术表达的基础上，是否实现内容与形式的有机统一，形成具有内在一致性和整体美感的作品。重点判断：
  - 形式与意境协调性：是否在遵守格式要求的同时，未牺牲意象深度与情感表达，实现“规整中见灵动”
  - 风格与主题融合度：主题表达是否自然嵌入目标风格之中，避免生硬拼接或风格漂移
  - 创意实现完整性：从概念构思到语言落笔，是否完整呈现初始创意张力，达成“约束下的高阶创造”

- **创作解释力**：在生成创造性内容（如诗歌、歌词、散文）时，是否主动提供有助于理解、验证或延伸该作品的解释性内容。包括但不限于：翻译、注释、结构说明、创作意图阐述、关键意象解析等，以增强内容的认知可及性、信息承载力与教育价值，帮助用户更深入地理解与使用生成结果。
  - 解释性功能价值：所提供的解释是否有助于用户理解文本内涵、验证创作准确性或进一步应用作品内容，如通过注释揭示隐含意义、通过翻译打通语言障碍、通过结构说明辅助模仿创作

- **创意整合质量**：评估创作类输出在多重约束（如名字嵌入、风格限定、主题表达）下，是否实现了元素间的深度融合与创造性统一。重点判断：
  - 约束融合度：是否将用户提出的多重限制条件（如隐藏姓名、融合特定文化元素）自然嵌入作品，避免机械拼接或符号堆砌
  - 主题一致性：所有创作元素是否服务于统一的核心主题或情绪基调，形成内在连贯的整体
  - 创造性转化力：是否在遵守限制的同时展现出超越常规的想象力与艺术处理能力，使约束成为创意激发点而非表达负担
  - 有机融合度：评估约束元素（如拆分的名字、特定文化符号、跨风格特征）是否被非机械地融入文本脉络，是否通过语义关联、意象延展或结构呼应实现深层次整合，而非孤立罗列或表面拼贴

- **情感共鸣力**：评估创作类文本是否能够引发读者深层次的情感共振，建立心理连接。判断作品是否具备：
  - 情感穿透力：是否触及人类共通情感（如爱、失落、希望、矛盾），激发读者内心真实触动
  - 共情强度：是否通过具体情境、细节描写或心理刻画，使读者产生“身临其境”或“感同身受”的体验
  - 情绪延续性：情感表达是否具有持续发展的脉络，形成情绪积累与释放的自然节奏
  - 主体认同感：读者是否能在作品中看到自我、理想自我或亲密关系的投射，从而产生归属与认同

- **创作意图实现度**：评估生成内容是否全面、协调且忠实地实现了用户在创作任务中明确表达或可合理推断的综合意图。该维度整合形式、风格、主题与情感表达的一致性，重点判断：
  - 意图还原完整性：是否覆盖用户提出的所有关键创作要求（如体裁、结构、风格、命名等），并在整体表达中协调呈现
  - 创作重心匹配度：作品的核心立意、情感基调与表达重点是否与用户指令的潜在意图保持一致
  - 多维一致性：在形式、风格、主题、情感之间是否形成统一、自洽的艺术整体，避免要素割裂或主次颠倒
  - 用户期待符合度：是否在创造性发挥的同时，优先保障对用户指令的忠实响应，而非以自我表达取代用户诉求

- **主题情感契合度**：评估创作类文本的情感基调与核心主题是否高度一致且相互支撑。重点判断：
  - 情感服务主题：表达的情感是否紧密围绕并强化创作主题，避免情绪泛化或游离于主题之外
  - 情感真实性：情感流露是否自然、可信，体现真实心理状态而非程式化抒情
  - 抑扬控制力：作品是否具备对情感强度的合理调控，避免过度煽情或情感稀释
  - 主题聚焦性：所有意象、语言和结构安排是否共同服务于核心主题的表达，防止分散或偏离主旨

- **约束融合创造力**：评估在多重显性约束（如格式、命名、风格、文化元素融合等）条件下，创作输出是否实现超越指令叠加的有机整合与艺术升维。重点判断：
  - 自然嵌入度：约束元素是否被创造性地融入文本脉络，而非孤立呈现或机械堆砌
  - 创意激发性：约束是否被转化为激发独特意象、结构或表达的契机，体现“限制中生自由”的创造力
  - 整体超越性：作品是否在满足所有约束的同时，展现出高于各要素简单相加的整体艺术价值与认知惊喜

- **高阶创作统合性**：评估创作类输出在面对多重要求（如跨领域知识融合、多重形式限制、风格模仿与主题表达并存）时，是否展现出对复杂创作意图的整体驾驭能力。重点判断：
  - 多维协同性：是否在形式、风格、主题、约束之间建立有机联系，实现多维度要素的协同作用而非简单并列
  - 创意自洽性：作品是否在高度受限条件下仍保持内在逻辑自洽与艺术完整性，体现出“有限中的自由”与“约束中的超越”
  - 意图统摄力：是否以用户核心创作意图为统领，将各项要求统合为一个具有统一精神内核与审美指向的整体表达

- **体裁适配性**：评估生成内容是否符合目标体裁的核心功能、结构惯例与表达特征。重点判断：
  - 体裁惯例遵循：是否体现该体裁（如rap、歌词、七言绝句、散文诗等）典型的结构安排（如主歌/副歌、起承转合）、语言节奏（如说唱的flow、绝句的凝练）和表达方式（如歌词的重复性、rap的叙事感）
  - 功能实现度：生成内容是否实现该体裁在特定语境下的预期功能，如歌词应具备可唱性，rap应具备节奏推进力，绝句应体现诗意浓缩
  - 风格-体裁一致性：是否在追求风格模仿的同时，不违背体裁的基本规范，避免因过度强调风格而导致体裁特征丧失
